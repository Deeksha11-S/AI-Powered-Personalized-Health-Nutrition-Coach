# 🌟 Personalized Health & Nutrition Coach

An AI-powered Health & Nutrition Recommendation System that generates personalized diet and fitness plans based on user information, health goals, and activity level.

Built using **Python**, **Machine Learning**, **Scikit-learn**, **Gradio**, and **Plotly** to provide an interactive web application for healthy lifestyle recommendations.

---

## 🚀 Features

- 👤 User Profile Creation
- 🔥 Automatic BMR & TDEE Calculation
- 🥗 Personalized Diet Recommendations
- 💪 Customized Fitness Plans
- 📊 Macronutrient Distribution Charts
- 📅 Weekly Workout Schedule
- 🤖 Machine Learning-based User Clustering
- 👥 Similar User Recommendation using KNN
- 📝 Feedback Collection System
- 🌐 Interactive Gradio Web Interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/Personalized-Health-Nutrition-Coach.git

cd Personalized-Health-Nutrition-Coach
```

---

### Create Virtual Environment

Windows

```bash
python -m venv .venv

.venv\Scripts\activate
```

Linux / Mac

```bash
python3 -m venv .venv

source .venv/bin/activate
```

---

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python main.py
```

The application will start locally and open in your browser.

Example:

```
Running on local URL:
http://127.0.0.1:7860
```
